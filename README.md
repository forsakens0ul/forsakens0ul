## 🎯 CS2 Settings (My Setup)

### 🎯 Crosshair
```cfg
cl_crosshairstyle 4;
cl_crosshairsize 2;
cl_crosshairthickness 0.5;
cl_crosshair_t 0;
cl_crosshairdot 0;
cl_crosshairgap -3;
cl_crosshair_drawoutline 0;
cl_crosshairusealpha 0;
cl_crosshaircolor 4;
```
